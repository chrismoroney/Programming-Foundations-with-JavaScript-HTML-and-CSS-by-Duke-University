Modifying an Image

by Chris Moroney

Aside from creating your own image, you can also modify an image with the power of Javascript!

I'm going to use my favorite teacher from Duke University, Professor Drew Hilton, as my example of modifying an image.

Here, we can see Professor Drew Hilton's portrait in a purple window frame.

The Algorithm

To start, I needed to create a border that would be the same number of pixels away from the edge of the photo on all sides. To do this, I needed to devise an equation that would put a border on the portrait that was no more than a certain number. This number I decided would be 10 pixels. This means on all sides, the border couldn't extend past ten pixels going up, down, left, or right into the portrait. Similary, I had to create a set amount of selective pixels to create a window-panel effect so that the image wouldn't extend into other parts of the portrait. I knew that in order to do this, I had to create an extenstion of pixels at the point of half of the width and half of the Height. I also wanted to make the line thicker, so I added and subtracted 5 to the base line in half of the Width and Height in order to be able to have the same type of line thickness as the border.

The Code 

// Edit existing image
var image = new SimpleImage("hilton.jpg");
for (var p of image.values()) {
    var w = image.getWidth();
    var h = image.getHeight();
    var midW = w/2;
    var midH = h/2;
    if (p.getX() < 10 || p.getY() < 10 || p.getX() >= image.getWidth()-10 || p.getY() >= image.getHeight()-10) {
            p.setRed(155);
            p.setBlue(200);
            p.setGreen(80);
        }
    if ( (p.getX() > (midW-5)) && (p.getX() < (midW + 5))){ 
        p.setRed(155);
        p.setBlue(200);
        p.setGreen(80);
    }
    if ( (p.getY() > (midH -5)) && (p.getY() < (midH + 5))){
        p.setRed(155);
        p.setBlue(200);
        p.setGreen(80);   
    }
}
print(image);
